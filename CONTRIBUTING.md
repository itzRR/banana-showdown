# Contributing to Banana Showdown

First off, thanks for taking the time to contribute!

## Development Setup
1. Clone the repository.
2. Install dependencies with `npm install` in both `client` and `server` folders.
3. Start the development servers.

## Pull Request Process
1. Ensure your code passes all lint checks and tests.
2. Update the README.md with details of changes to the interface, this includes new environment variables, exposed ports, useful file locations and container parameters.
3. You may merge the Pull Request in once you have the sign-off of at least one other developer.
